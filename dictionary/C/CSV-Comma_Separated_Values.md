# CSV: Comma-Separated Values

## Name

## Overview
- A delimited text format that uses a comma to separate values.
   - Each line of the file is a data record
   - Each record consists of one or more fields, separated by commas.
- Example:
  ```csv
  Year,Make,Model
  1997,Ford,E350
  2000,Mercury,Cougar
  ```
