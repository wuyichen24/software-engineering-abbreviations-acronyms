# FDD: Feature-Driven Development

## Name

## Overview
- An iterative and incremental software development process:
   - Driven from a feature perspective.
- FDD consists 5 basic activities:
   - Develop overall model
   - Build feature list
   - Plan by feature
   - Design by feature
   - Build by feature
